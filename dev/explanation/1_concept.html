Concept ​

The AIBECS (pronounced like the cool ibex if you have a french accent) is a new software written in Julia to easily create some marine biogeochemistry models in just a few commands.

AIBECS is not just a single model. It's a system that allows you to create a global steady-state biogeochemistry model with just a few simple commands. Basically, you just need to tell AIBECS which ocean circulation to use (from simple toy models of just a few boxes to more complicated global models of the circulation), what elements you want to model/track, and how they are converted into other (the net local sources and sinks of your model). Once the model is set up, chose some parameter values and you can run simulations.

AIBECS relies on many tools from linear algebra to run simulations and perform optimizations really fast.
