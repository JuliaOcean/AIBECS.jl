Groundwater discharge ​

In this tutorial we will simulate a fictitious radioactive tracer that is injected into the ocean by groundwater discharge. The global groundwater discharge dataset for 40,000 coastal watersheds from Luijendijk et al. (2020) is available from within the AIBECS. Once "born", our ficitious tracer decays with a parameter timescale τ as it flows through ocean basins.

The 3D tracer equation is:

[∂∂t+∇⋅(u+K∇)]x=sgw−x/τ

where ∇⋅[u−K∇] represents the ocean circulation transport. (Tracer transport operators are described in the documentation.) The source of the tracer is sgw, and x/τ is the decay rate.

In AIBECS, we must recast this equation in the generic form

[∂∂t+T(p)]x=G(x,p).

We start by telling Julia that we want to use the AIBECS and the OCIM2 transport matrix for the ocean circulation.

julia
using AIBECS
using JLD2                  # required by `OCIM2.load`

because the source is given by the mass (discharge × concentration) divided by the volume of each box:

julia
function s_gw(p)
    @unpack C_gw = p
    return @. gws_grd * C_gw / v
end

We then write the generic G function, which is

julia
G_gw(x,p) = s_gw(p) - decay(x,p)

We specify some initial values for the parameters and also include units.

julia
import AIBECS: @units, units
import AIBECS: @initial_value, initial_value

Or we can increase the decay timescale (×10) and decrease the groundwater concentration (÷10) to get a different (more well-mixed) tracer distribution:

julia
p = GroundWatersParameters(τ = 200.0u"yr", C_gw = 0.1u"mol/m^3")
prob = SteadyStateProblem(F, x, p)
sol_τ50 = solve(prob, CTKAlg()).u * u"mol/m^3"
